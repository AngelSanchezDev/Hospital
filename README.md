# 🏥 Hospital Management System

Program template developed in Python, focused on hospital management and administration.

## 🚀 Features
* **MVC Architecture:** Clear separation of logic with Controllers, Models, and Views.
* **Database:** Complete relational structure (includes `.sql` scripts and `.bak` backups).
* **Reports:** Dedicated module for generating system reports.

## 🛠️ Technologies Used
* **Language:** Python 3
* **Database:** SQL Server (or the database used)
* **Design Pattern:** Model-View-Controller (MVC)

## 📦 Project Structure
* `/controllers`: Control logic and data flow.
* `/models`: Connection, queries, and business logic.
* `/views`: User interface and screens.
* `/Database`: Database installation and backup scripts.

# 🏥 Sistema de Administración de Hospital

Modelo de programa desarrollado en Python enfocado en la gestión y administración hospitalaria.

## 🚀 Características
* **Arquitectura MVC:** Separación limpia de la lógica con Controladores, Modelos y Vistas.
* **Base de Datos:** Estructura relacional completa (scripts `.sql` y respaldo `.bak` incluidos).
* **Reportes:** Módulo dedicado para la generación de reportes del sistema.

## 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Python 3
* **Base de Datos:** SQL Server (o la que uses)
* **Patrón de Diseño:** Model-View-Controller (MVC)

## 📦 Estructura del Proyecto
* `/controllers` - Lógica de control y flujo de datos.
* `/models` - Conexión, consultas y lógica de negocio.
* `/views` - Interfaz y pantallas del usuario.
* `/BaseDeDatos` - Scripts de instalación y copias de seguridad de la BD.
